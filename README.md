# 📡 쇼츠 뉴스 메이커

대전·세종·충남 지역 뉴스를 Claude AI로 자동 생성하는 유튜브 쇼츠 제작 도구

## 🚀 배포 주소
https://[내 GitHub 아이디].github.io/shorts-news-maker/

## 📱 주요 기능
- 🟠 Claude AI로 대전권 뉴스 자동 생성
- ✏️ 헤드라인 / 본문 자동 작성
- 🎨 6가지 뉴스 템플릿 디자인
- 🔊 Google Neural2 / Gemini TTS 고품질 음성
- 📤 공유 및 텍스트 복사

## 🔑 필요한 API 키
| 키 | 용도 | 발급처 |
|---|---|---|
| Anthropic API Key | 뉴스 생성, 헤드라인, 대본 | console.anthropic.com |
| Google Cloud API Key | Neural2 TTS 음성 | console.cloud.google.com |
| Gemini API Key | Gemini 2.5 TTS 음성 | aistudio.google.com |

> API 키는 브라우저 localStorage에만 저장되며 외부로 전송되지 않습니다.
